# MyFrame

MyFrame is a multilingual smart frame app starter built with Next.js + TypeScript.

## Languages

- English (`/en`)
- Mandarin Chinese (`/zh`)
- Spanish (`/es`)
- French (`/fr`)

## Run Locally

This machine currently has an old system Node version. For this project, use Node 20:

```bash
export PATH="/home/h/.local/node20/node-v20.19.0-linux-x64/bin:$PATH"
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000). The app redirects to `/en` by default.
